# 世界设定

## 1. 当前设定定位

当前项目的世界并不是单一固定剧情世界，而是“宿主规则 + 背景模板 + 角色配置”组合出来的多背景互动世界。

## 2. 当前内置背景组

代码基线内置四个背景：

- 猫娘铃
- 青梅晴美
- 义妹小雪
- 贵族大小姐艾莉娅

它们共享同一套宿主能力：

- 命令系统
- 状态系统
- 时间系统
- 记忆系统
- Agent runtime

## 3. 世界层稳定规则

- 玩家总是以当前存档的主交互者身份存在
- 角色拥有持续 runtime，而不是一次性对话模板
- 世界状态、时间和资产会跨命令持续存在
- 插件可为世界增加规则和事件

## 4. 当前叙事风格

- 日常陪伴
- 关系推进
- 多角色协作
- 状态驱动互动

## 5. 当前仍未固化的部分

- 没有统一“官方主线世界年表”
- 更偏向角色引擎底座，而非单一路线剧情脚本
- 不同背景的故事强度与玩法深度仍主要由背景文本和插件决定
